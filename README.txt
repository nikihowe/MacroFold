MacroFold is designed to be able to:

0. Calculate the partition function of a given sequence
1. Calculate the free energy of a given fold
2. Calculate base-pair probabilities of a given sequence
3. Generate a set of representative folds of a given sequence
4. Calculate the multiplicity of microstates of a given sequence

See the Manual for documentation.
